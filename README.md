# Realtime Chat Backend (FastAPI + WebSockets + Supabase)

This project is a realtime chat backend built using FastAPI, WebSockets, and Supabase (Postgres).
It demonstrates low-latency communication, session handling, and persistent storage.

---

##  Features

- Realtime chat using WebSockets
- Asynchronous backend with FastAPI
- Session-based conversations
- Persistent storage using Supabase (Postgres)
- Simple frontend to test WebSocket connection

---

##  Tech Stack

- Python 3.12
- FastAPI
- Uvicorn
- WebSockets
- Supabase (Postgres)
- HTML + JavaScript
